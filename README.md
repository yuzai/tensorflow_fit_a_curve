### 代码功能
三种方法实现曲线拟合,[demo](https://blog.maxiaobo.com.cn/tensorflow_fit_a_curve/dist/)

### 使用方法
1. git clone git@github.com:yuzai/tensorflow_fit_a_curve.git
2. cd  tensorflow_fit_a_curve
3. yarn 
4. yarn watch
5. 打包生成使用yarn build
